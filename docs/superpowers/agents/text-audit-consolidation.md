## Task
Read all 7 individual audit JSON files and consolidate them into a comprehensive markdown report with executive summary, priority action items, and section breakdowns.

## Input Files
Read from `.omc/audit-outputs/`:
1. `registration-audit.json`
2. `public-pages-audit.json`
3. `admin-audit.json`
4. `technique-library-audit.json`
5. `navigation-ui-audit.json`
6. `forms-inputs-audit.json`
7. `error-success-audit.json`

## Method

1. **Read all 7 JSON audit files** from `.omc/audit-outputs/`

2. **Parse and analyze** the findings:
   - Count total issues across all sections
   - Summarize severity breakdown (high/medium/low)
   - Identify most affected files
   - Extract common patterns of AI-generated text

3. **Generate priority action items**:
   - **High Priority**: Issues that significantly impact user experience or brand voice
   - **Medium Priority**: Issues that should be addressed in next sprint
   - **Low Priority**: Minor improvements for polish

4. **Create comprehensive report** in markdown format
